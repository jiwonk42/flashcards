  • Javascript language that responds to user
  • Operators +*-/Javascript sees these and performs math
  • Variables always put var to name a variable. variables store the values to use in other functions and places.
  • Variable naming conventions var name = value. needs to be descriptive and specific
  • Functions do a set of statements multiple times
  • Methods *pre-made??* functions attached to objects. syntax: object.method(functions)
  • Arguments values that we put into the place of parameters in functions
  • Parameters placeholders in functions
  • Return output of function. the computer stores it so that we can use the returned value in other places
  • Chaining method calling multiple methods on a value one after the other
  • Strings words or something that JavaScript interprets as words. Within quotation marks
  • Booleans true or false values. not within quotation marks
  • Undefined variable hasn't been initialized or stored.
  • NaN a type of number. stands for not a number
  • Escape backslash that allows a character to be a part of a string
  • Alert jQuery function that opens a window to alert user of something. appears with a clickable box that says "ok"
  • Comments tell anyone looking at the code what something in the code does. especially helpful for functions. in JavaScript,
  • jQuery JavaScript library. must be linked to document in html file
